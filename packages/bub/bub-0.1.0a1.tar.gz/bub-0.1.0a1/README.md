Bub is a CLI tool that helps you build things, with AI.

## Quick Start

```bash
# Set up your API key
export BUB_API_KEY="your-api-key-here"

# Start interactive chat
bub chat

# Run a single command
bub run "Create a hello world script"

# Get help
bub --help
```
