This module allows you to display the Product Stickers in the Delivery
Slip reports.

**Table of contents**

.. contents::
   :local:

Installation
============

You must install the ``product_sticker`` module from [Product
Attribute](https://github.com/OCA/product-attribute) repository.

Usage
=====

To use this module, you need to:

1. You must go to the Picking Type that you want to show the Product
   Stickers and choose the desired position of the stickers on the field
   *Show Product Stickers*.
2. You can also change the position or disable it on each Picking by
   hand if you wish. Regardless if your Picking Type has Show Stickers
   enabled or not.
